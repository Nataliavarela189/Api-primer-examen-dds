# 🏍️ CRUD de Motos (Node.js + Express + Fetch)

Aplicación web simple que implementa un CRUD de motos, conectando un backend en Node.js con un frontend en JavaScript mediante peticiones HTTP.

---

## 🧩 Descripción

Este proyecto consiste en una aplicación básica que permite:

* Crear, leer, actualizar y eliminar motos
* Conectar frontend y backend mediante requests HTTP
* Practicar el flujo completo cliente-servidor

---

## 🚀 Funcionalidades

* Listado de motos
* Alta de nuevas motos
* Edición de datos
* Eliminación de registros
* Comunicación entre frontend y backend usando Fetch API

---

## 🏗️ Arquitectura

* Backend desarrollado con **Express**
* Frontend en **HTML + JavaScript**
* Comunicación mediante **API REST**

---

## 💻 Tecnologías

### Backend

* Node.js
* Express

### Frontend

* HTML
* JavaScript
* Fetch API
